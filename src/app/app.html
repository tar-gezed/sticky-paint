Aide & Raccourcis

Outils & Dessin

Sélectionnez un outil (Crayon, Formes, Gomme) puis dessinez sur le canevas en maintenant le clic.

  • Couleur 1 (C1): Couleur du tracé / contour.
  • Couleur 2 (C2): Couleur de remplissage (ou Gomme).
  • Taille: Épaisseur des traits ou de la gomme.

Raccourcis Clavier

Annuler: Ctrl + Z
Rétablir: Ctrl + Y
Copier: Ctrl + C
Coller: Ctrl + V
Couper: Ctrl + X
Supprimer: Suppr

Astuces

  • Faites glisser une image locale sur le canevas pour l'importer.
  • Glissez-déposez les calques dans le panneau pour réorganiser leur ordre.
  • Double-cliquez sur les textes pour les éditer directement.
